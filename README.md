# Text Analyzer

This is my first project created for the Engeto Online Python Academy. It is a simple console application written in pure Python. The goal of this project was to practice basic programming concepts such as working with strings, dictionaries, lists, loops, and conditions.

## Features

The program simulates a basic login process and then analyzes one of three predefined texts. It calculates and prints the following statistics:
* Total number of words.
* Number of titlecase, uppercase, and lowercase words.
* Number of numeric strings and their total sum.
* A text-based bar chart showing the frequency of different word lengths (excluding punctuation).

## How to Run

No external libraries are required. You just need to have Python installed on your computer.

2. Run it in your terminal or command prompt:
